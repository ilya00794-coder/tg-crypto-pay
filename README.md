# Crypto Pay Telegram Mini App 💸

## Быстрые криптоплатежи в Telegram

### 🚀 Возможности
- Моментальная оплата криптовалютой
- 6% комиссии платформы
- Безредиректная интеграция
- Поддержка USDT, TON и других криптовалют

### 🔧 Стек
- Backend: FastAPI (Python)
- Frontend: React + TypeScript
- Платежный провайдер: 2328.io

### 🛠 Установка
1. Клонировать репозиторий
2. Настроить `.env`
3. Установить зависимости
4. Запустить сервисы

### 📦 Компоненты
- `/backend` - серверная логика
- `/frontend` - Mini App интерфейс
- `/telegram-bot` - Telegram Bot

### 💡 Как работает
1. Мерчант генерирует ссылку оплаты
2. Клиент оплачивает в один клик
3. Деньги мгновенно поступают продавцу
